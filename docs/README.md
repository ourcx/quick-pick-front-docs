## 项目概览

**庆快校园** 是一个专注于校园生活服务的前端项目集合。我们致力于通过现代化的前端技术栈，为学生提供便捷、流畅的数字化校园体验。

### 核心技术栈

- **框架**: [Vue 3](https://vuejs.org/) + [Uni-app](https://uniapp.dcloud.net.cn/)
- **状态管理**: [Pinia](https://pinia.vuejs.org/)
- **网络通信**: Axios + SSE (Server-Sent Events)
- **文档方案**: [VuePress 2](https://vuejs.press/)

# 通信方案

本分类介绍小程序中各种通信方式的实现方案。

## 包含内容

- **SSE（Server-Sent Events）** - 服务端推送事件
- 轮询方案
- WebSocket方案

## 选择指南

| 通信方式 | 特点 | 适用场景 |
|---------|------|---------|
| **SSE** | 单向推送、轻量级、基于HTTP | 实时通知、消息推送 |
| **轮询** | 简单、兼容性好 | 低频更新、简单场景 |
| **WebSocket** | 双向通信、实时性强 | 聊天、游戏、高频交互 |
